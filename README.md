The integration of disparate data sources CSV, JSON, and SQL into a single, 10,000 row dataset for business intelligence is the main focus of this project, which illustrates a full Data Engineering and Analytics pipeline. The analysis reveals important performance metrics like the impact of Gold membership, citywise revenue distribution, and seasonal order trends by using intricate Left Joins and data transformation techniques. The goal is to offer a data-driven basis for maximizing marketing tactics and operational effectiveness in a cutthroat food delivery market.
